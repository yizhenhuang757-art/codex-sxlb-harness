# Skill Clans

Clan 是 family 上面的人读入口层。它的目的不是增加流程，而是减少选择成本。

## 规则

- clan 数量不超过 10 个。
- Readme 是总目，必须列出所有 active skill。
- clan 文档是主索引文档；family 只作为中间分组和 SXLB 召回单位。
- clan 不授予执行权，执行仍看 family allowlist/mapping 和具体 `SKILL.md`。
