# v0.1.0 Public Edition

Initial public edition of the `sxlb` Codex harness under the target project path `HYZ/codex-sxlb-harness`.

## Includes

- `sxlb` skill package with `SKILL.md`, `MODE.md`, roles, protocols, templates, hooks, scripts, tests, and examples.
- Portable workflow layer for 案卷体系, 国史馆, 起居郎, 翰林院, strict external screening, forced status boards, and review gates.
- Intervention surface for `会审`, `侍讲官 <问题>`, `重审`, `录案`, `事件簿`, `国史馆`, `翰林院`, `起居郎`, and `退朝`.
- Optional conversion path from harness mode into multi-agent / multi-skill orchestration through `翰林院` reference-pool admission, `吏部` appointment, `尚书省` dispatch, and `门下省` review.
- Bundled companion skill `sxlb-agent-dispatch-check`.
- Public-safe skill inventory behavior that uses placeholders unless explicitly asked to scan local skills.
- MIT license draft and GitHub Actions workflow draft.

## Excludes

- Personal case history, worklogs, vault contents, private memory stores, plugin caches, browser profiles, credentials, and account data.
- TOEFL, TickTick, Zotero, communication, internal wiki, and other personal-domain contents.
