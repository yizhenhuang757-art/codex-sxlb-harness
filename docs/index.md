# Govern work without losing momentum

SXLB is a text-first governance harness for work that needs clear proposals, independent review, accountable execution, and durable records. Start with the smallest useful workflow; read the historical model only when you want to understand why the sequence is designed this way.

> **Start here:** [Install SXLB]({{ '/quickstart/' | relative_url }}) and create one governed case. The public package includes empty, portable templates rather than a copy of anyone's private operating history.

## What you can expect

- A visible chain from request to proposal, review, execution, verification, and closure.
- A status board and case record that make current state and accountability easy to inspect.
- Optional multi-agent or multi-skill work only when coordination is worth its cost.
