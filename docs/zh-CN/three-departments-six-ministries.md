# 三省六部

三省六部制是中国古代重要行政模型的常用概称，尤其常与隋唐联系，并在后世王朝中被调整和继承。它之所以适合在这里介绍，是因为它使一个治理问题变得可见：政策形成、审议、行政统筹和专门工作不应坍缩为一次不受检查的行动。

> 本页提供的是便于使用的制度模型，并不主张每个朝代都以不变的职权使用相同机构。机构名称、品级、隶属关系和实际权力会随时代发生显著变化。

## 三省：区分环节的链条

| 机构 | 常见的高层职能概括 | 治理逻辑 |
| --- | --- | --- |
| 中书省 | 起草、拟定诏令与政策文书 | 方案应在执行前被明确提出 |
| 门下省 | 审议、复核、谏诤，并可通过封驳退回有问题的草案 | 方案需要获得独立提出异议和纠正的机会 |
| 尚书省 | 在经典模型中统筹行政，并总领六部 | 获准的决定仍需要可追责的执行与协调 |

重点不在于把它理解成机械的线性官僚制。实际政务还涉及皇帝、朝议、秘书性机构、监察机构、军事体系、地方政府以及不断变化的朝代安排。三省模型持久的分析价值，在于它区分了起草、复核和执行。

## 六部：专门化的行政领域

在常见模型中，六部隶于尚书省。它们的职责也随时代而变化，但下列概括可以作为可靠的入门。

| 部 | 核心历史职掌 | 常见关切 |
| --- | --- | --- |
| 吏部 | 官员任用、考课、升迁与行政人事程序 | 谁具备资格、被任命、负责或被调任 |
| 户部 | 户籍、人口、土地与赋税管理，以及国家财政和粮储 | 人口、收入、财政能力和资源 |
| 礼部 | 国家礼仪、教育、科举，以及外交和朝贡接待 | 正统性、礼仪、学术、考试和正式关系 |
| 兵部 | 军事行政、武官人事、后勤、交通及相关文书 | 动员、军事行政、协调与国防支持 |
| 刑部 | 刑法、司法行政、刑罚与案件复核 | 法律标准、程序、责任与惩戒 |
| 工部 | 公共工程、营造、水利、技术生产及相关劳役、物料管理 | 基础设施、建设、工程和物质性执行 |

六部并不垄断这里所列的全部职能。例如，司法事务还涉及大理寺、御史台等机构，军事指挥也可能通过其他机构组织。此表的价值在于展示行政责任的分化，而不是抹除这些制度差异。

## 为什么这一模型至今仍有启发

它提出四个持久的问题：

1. 方案是否已经清晰到足以被评估？
2. 在行动成本变高之前，是否有人能以不同角色对它提出挑战？
3. 方案获准后，谁来统筹执行？
4. 工作的各部分由哪个专门责任边界承担，又留下什么记录？

SXLB 将这些问题借为当代知识工作的治理模式。它不会把智能体当作字面意义上的帝制官员，也不宣称与历史制度一一等同。请继续阅读[SXLB 映射与设计逻辑]({{ '/zh-CN/sxlb-mapping/' | relative_url }})，了解准确的现代转换。
